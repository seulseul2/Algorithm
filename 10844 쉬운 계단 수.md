__10844 쉬운 계단 수__

```python
n = int(input())

lst = [1] * 10
lst[0] = 0
cnt = 1

while n > cnt:
    dp = [0] * 10
    for i in range(10):
        if i != 0:
            dp[i-1] = (dp[i-1] + lst[i]) % 1000000000
        if i != 9:
            dp[i+1] = (dp[i+1] + lst[i]) % 1000000000
    lst = dp
    cnt += 1
print(sum(lst) % 1000000000)
```

- 예전에 풀어봤던 dp 문제랑 비슷했다.
- 계산 식
  - 끝자리가 0인 경우 -> 1자리가 늘어났을 때, 1로 끝나는 수에 추가
  - 끝자리가 9인 경우 -> 1자리가 늘어났을 때, 8로 끝나는 수에 추가
  - 나머지 -> +1과 -1로 끝나는 수에 추가
