__1436 영화감독 숌__

```python
n = int(input())
cnt = 0
x = 666

while 1:
    if '666' in str(x): #문자열로 바꿔서 이 안에 666이 있을 경우
        cnt += 1 # 순번을 +1 더해준다. 즉, n=666, cnt=1 / n=1666, cnt=2
    if cnt == n:
        print(x)
        break
    x += 1 # 계속해서 숫자를 올리면서 체크
```
